# LOBF-Calculator
poly_est: Calculates polynomial equation parameters such that, for a sequence of numbers, the next in sequence is a number of your choosing.

exp_est: Does the same but with exponential equations; not finished.

stats: Returns info about a sequence of numbers.

funcs: Allows math with lists of numbers.
